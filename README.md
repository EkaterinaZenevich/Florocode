# Флорокод

Локальная статическая версия прототипа для анализа флоронимов в корпусе русской женской прозы первой половины XIX века.

## Быстрый запуск

На Windows можно дважды щелкнуть `start_windows.bat`.

Если Python установлен:

```bash
python -m http.server 8765
```

После запуска открыть в браузере:

```text
http://127.0.0.1:8765/
```

## Публикация на GitHub Pages

1. Создать новый репозиторий на GitHub.
2. Загрузить в него все файлы из этой папки.
3. В настройках репозитория открыть `Pages`.
4. Выбрать публикацию из ветки `main`, папка `/root`.
5. GitHub выдаст публичную ссылку на сайт.

## Состав

- `index.html` — главная страница.
- `styles.css` — оформление.
- `app.js` — логика интерфейса.
- `data.js` — корпус и найденные флоронимы.
- `meanings.js` — карточки значений по Шарафадиной, Ознобишину и корпусным наблюдениям.
