# Флорокод

Локальная статическая версия прототипа для анализа флоронимов в корпусе русской женской прозы первой половины XIX века.

## Состав

- `index.html` — главная страница.
- `styles.css` — оформление.
- `app.js` — логика интерфейса.
- `data.js` — корпус и найденные флоронимы.
- `meanings.js` — карточки значений по Шарафадиной, Ознобишину и корпусным наблюдениям.
